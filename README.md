## Environments

To Install the dependencies and devDependencies in following environments

**For Development**

```sh
npm install
npm run dev
```


**For local production Test**

```sh
npm install
npm build
npm run start:prod
```

**For Production Environment**

```sh
npm install
npm build
npm run start:prod
```
